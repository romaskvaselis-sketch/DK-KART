# Sample telemetry files

This folder contains sample CSV exports from various karting telemetry devices,
used for testing the parsers.

All samples are anonymized (driver names removed).

## Contribute

If you have a sample from a device not yet supported, please submit it via
GitHub issue or PR. See [CONTRIBUTING.md](../CONTRIBUTING.md).
